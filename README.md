# Sistema de Ferretería

Este proyecto consiste en el desarrollo de un sistema para la gestión de una ferretería.

## Objetivo
Facilitar el control de productos, clientes y ventas dentro de una ferretería.

## Funcionalidades
- Registro de productos
- Gestión de clientes
- Control de ventas
- Emisión de comprobantes

## Herramientas
- Base de datos
- Lenguaje de programación (por definir)
- GitHub para control de versiones
